# Premium Scroll Animation System (BoilerLab Style)

## Overview

This project implements a premium scroll-driven website inspired by
BoilerLab, Apple, Stripe, and Linear.

## Tech Stack

-   Next.js 15
-   React 19
-   TypeScript
-   Tailwind CSS
-   GSAP + ScrollTrigger
-   Lenis
-   Framer Motion
-   Three.js / React Three Fiber
-   SplitType
-   HTML5 Canvas

## Features

-   Smooth scrolling
-   Scroll-triggered animations
-   Sticky/pinned sections
-   Hero animations
-   Horizontal scrolling
-   Card stacking
-   Text reveal animations
-   Parallax effects
-   Mouse interactions
-   Responsive design
-   Accessibility support
-   Performance optimized

``` bash
npm install gsap @studio-freight/lenis split-type framer-motion three @react-three/fiber @react-three/drei lottie-react
```

Create a premium, cinematic, Apple-style scrolling experience with
